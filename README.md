## パッケージのインストール
```
npm install dotenv  
npm install mongoose  
```

## MongoDB
.env ファイルを作成し、以下を書き込む
```
MONGO_URL="mongodb+srv://username:password@cluster0.17g8u.mongodb.net/TODO_App?retryWrites=true&w=majority&appName=Cluster0"
```
